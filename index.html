Sobre o site

Um aplicativo onde os usuários podem criar enquetes onde os eleitores podem sugerir quaisquer jogos disponíveis no IGDB para jogar em uma transmissão ou reunião de jogos. IGDB (Internet Game Data Base) possui uma API útil para obter jogos e seria necessário implementar algum tipo de pesquisa AJAX para isso.

Serviços

Oferecemos um serviço onde jogadores podem se juntao ao GameFinityHub para compartilhar ideias e ajudar outros jogadores a decidirem qual próximo jogo jogar

Criação de enquetes

Os jogadores podem criar suas próprias enquetes para que outros possam votar ou adicionar jogos nelas

Votar em enquetes

Você pode ver enquetes ja criadas por outros jogadores para dar a sua opinião

Resultados

O GameFinityHub disponibilizará o resultado da sua enquete para que você possa analisar as opções e votações, com uma lista em ranking dos mais votados!
